# **App Name**: BusETA

## Core Features:

- Route Visualization: Display a map showing the bus route, current bus location, and upcoming stops.
- ETA Prediction: Display the estimated time of arrival (ETA) for the next bus stop based on the bus's current location and route.
- Driver Info: Allow drivers to set their information for the day, including name and bus route.
- Location Tracking: Enable drivers to transmit their location data to the server.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to evoke feelings of reliability and safety.
- Background color: Light Gray (#EEEEEE), nearly white, providing a clean and readable interface.
- Accent color: Teal (#009688) to highlight key information like ETA and active stops.
- Body and headline font: 'PT Sans', sans-serif, combining modern and warmth.
- Use clear and recognizable icons for bus stops, location markers, and other key elements.
- Employ a clean and straightforward layout with intuitive navigation, making it easy for both drivers and passengers to use.
- Implement subtle animations, such as smooth map updates and ETA transitions, to enhance the user experience.
